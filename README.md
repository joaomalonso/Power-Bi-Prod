# Power-Bi-Prod
Painel de Análise de Casamentos com Power BI.

Descrição: O projeto consiste em desenvolver um painel de análise utilizando a plataforma Power BI para uma empresa que realiza casamentos. O objetivo principal é fornecer à empresa uma visão clara e abrangente dos dados relacionados aos casamentos que ela organiza, permitindo uma melhor compreensão dos principais indicadores e facilitando a tomada de decisões estratégicas.

O painel será criado a partir da integração de diversas fontes de dados, como planilhas de orçamentos, listas de convidados, datas e locais dos eventos, entre outros. Essas informações serão extraídas, transformadas e carregadas no Power BI, utilizando suas capacidades de modelagem de dados e preparação de dados.

Além disso, o painel poderá conter filtros e segmentações que permitirão à empresa explorar os dados de maneira personalizada. Por exemplo, será possível filtrar por período específico, por localidade ou vendedor, a fim de identificar tendências, padrões ou possíveis oportunidades de negócio.
